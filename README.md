# File-Transfer-app

DaneR file share is an Android LAN file transfer app.

Features:
- Browse Windows shared folders over SMB.
- Browse local phone storage.
- Copy, paste, delete and open files.
- Camera capture directly to a selected PC folder.
- Phone-to-phone transfer over local WiFi/hotspot.
- Transfer progress with circular percent indicator.
- Shows available free storage for connected devices.

This repository contains the open-source Android project files.
